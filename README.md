# OpenLiveReplay

### Objective
Performant cross-platform multi-track live video recorder & replay
### Target audience
For now, event safety and sports judging. Later hopefully small scale video live productions & sports events.
### Status
Prototyping state. OSX & iOS builds work with at least SRT recording being reliable, nearing MVP.

## Install
Checkout source & use QT Creator to build. Build phase should setup FFMPEG & rtmidi for OSX & iOS deployment.
